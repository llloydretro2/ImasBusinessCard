# ImasBusinessCard · 偶像大师 公式化名片模板

为 283 事务所（学园偶像大师 / 闪耀色彩）制作人制作**公式化名片**的 HTML 模板。
原始设计 by **AyasanaP**。

标准名片比例 91mm × 55mm（910 × 550px），正面 / 背面各一张，浏览器打开即可预览、截图或打印。

## 目录结构

```
公式化名片模板/
├── 模板/                    ← 空白模板（只读，复制用，勿直接改）
│   ├── 名片正面.html
│   └── 名片背面.html
│
├── 名片素材/                ← 共享素材库（大家共用，引用而非复制）
│   ├── 企划LOGO/            ← 事务所 / 企划 logo（如 283事务所logo.png）
│   ├── 组合LOGO/            ← 各组合 logo（如 COMETIK.png）
│   └── 签名/                ← 角色签名，按作品分类（Shiny Colors 等）
│
└── 名片存档/                ← 每张名片一个独立文件夹，永久存档、互不影响
    └── ShinjoP_铃木羽那/     ← 示例
        ├── 正面.html
        ├── 背面.html
        └── 素材/
            ├── 立绘.jpg      ← 本卡专属角色图
            └── 签名.webp     ← 本卡专属签名（也可改用共享库里的官方签名）
```

## 新建一张名片（三步）

1. **复制** `模板/` 里的 `名片正面.html`、`名片背面.html` 到
   `名片存档/你的名字_担当偶像/` 下，改名为 `正面.html`、`背面.html`。
2. 在同目录建一个 `素材/` 文件夹，放入 **`立绘.jpg`**（角色图）。
   签名可放 **`素材/签名.webp`**，或直接引用共享库
   `../名片素材/签名/作品/角色.png`。
3. 打开这两个 HTML，把 `{{...}}` 占位符和带 ①②③ 注释的地方改成自己的信息
   （偶像名、制作人名、QQ / B站、组合 logo 与事务所 logo 文件名）。

完成后用浏览器打开即可预览。**每张名片都是独立文件夹，新建时无需删除或替换任何共享素材。**

## 路径速记

| 素材类型 | 存放位置 | HTML 里的引用路径 |
|---------|---------|------------------|
| 角色立绘 | 本卡 `素材/` | `素材/立绘.jpg` |
| 角色签名 | 本卡 `素材/` 或共享库 | `素材/签名.webp` 或 `../名片素材/签名/...` |
| 组合 logo | 共享库 `组合LOGO/` | `../名片素材/组合LOGO/xxx.png` |
| 事务所 logo | 共享库 `企划LOGO/` | `../名片素材/企划LOGO/283事务所logo.png` |

> 注意：文件夹名 `组合LOGO`、`企划LOGO` 为**大写**，引用时大小写要一致
> （macOS 不区分大小写，但上传 GitHub / Linux 会区分，写错图片会不显示）。
